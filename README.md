# ai-desk

AIにプログラムを書かせるための土台。

人間がやるのは「やりたいこと」を自然言語でAIに伝えることだけ。コードの読み書き・修正・テストはAIが担当します。

> [ブラウザでデモを触る](https://aoyamarito.github.io/ai-desk/actions/index.html)

---

## どんな人に向いているか

- AIに任せて何かを作りたいけれど、毎回コードがバラバラになって続かない人
- ゲームのルール・業務ロジック・アプリのUIなど「条件分岐が多い」ものをAIに作ってほしい人
- AIに頼んだあと「ここおかしい」と気付いたとき、自分で直すのではなくAIに直させたい人

複雑な状態の組み合わせバグを構造的に潰す仕組みが入っているので、AIが書いたコードが「途中まで動いて崩壊する」状況になりにくくなっています。

---

## 使い方

人間がコマンドを叩く必要はありません。AIに自然言語で頼むだけです。

### 何かを作りたいとき

AIにこう言ってください:

```
AI_ONBOARDING.md を読んでから、[作りたいもの] を作ってください。
```

例:

```
AI_ONBOARDING.md を読んでから、
じゃんけんゲームをブラウザで動くように作ってください。
```

### バグを直したいとき

```
AI_ONBOARDING.md を読んでから、
ai-eyes を起動して、[バグの現象] を直してください。
```

サーバーの起動・観測・修正・確認まで AI 側でやってくれます。

### 既存ファイルに手を入れたいとき

```
AI_ONBOARDING.md を読んでから、
app.js の [機能名] を [変更内容] してください。
```
